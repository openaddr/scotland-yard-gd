● 根据多个来源交叉验证，以下是 Scotland Yard（苏格兰场）
  的完整规则。我之前记忆中有几处小错误（比如揭露回合数），以下是修正后的准确版本：

---
  基本信息

  - 人数：3–6 人（1 人扮演 Mr. X，其余扮演侦探）
  - 时长：约 45–90 分钟
  - 年龄：10+
  - 设计者：Werner Schlegel、Karl-Heinz Schmiel、Fritz Ifland 等
  - 发行：Ravensburger，1983 年，获当年 Spiel des Jahres（德国年度桌游奖）

---
  游戏设置

  1. 分配角色：选定 1 人扮演 Mr. X，其余为侦探。如果玩家人数少于
    5，可以一人操控多个侦探棋子，但每个侦探的票必须分开管理。
  2. 抽取起点：Mr. X
    随机抽一张起点牌，将棋子放在对应编号站点上（不让侦探看到）。每位侦探也各抽一张起点牌，公开放置棋子。
  3. 分发交通票：

    - Mr. X 获得：出租车 ×4、公交车 ×3、地铁 ×3、黑票 ×5、双倍移动票 ×2
    - 每位侦探获得：出租车 ×10、公交车 ×8、地铁 ×4

---
  三种交通方式

  移动时必须使用两站之间连线颜色对应的票：

  ┌──────────┬──────────┬────────────────────────┐
  │ 交通工具 │ 路线颜色 │          特点          │
  ├──────────┼──────────┼────────────────────────┤
  │ 出租车   │ 黄色细线 │ 短距离，覆盖最广       │
  ├──────────┼──────────┼────────────────────────┤
  │ 公交车   │ 绿色粗线 │ 中距离，连接更多站点   │
  ├──────────┼──────────┼────────────────────────┤
  │ 地铁     │ 红色虚线 │ 长距离，可快速穿越伦敦 │
  └──────────┴──────────┴────────────────────────┘

  移动规则：
  - 每次只能移动到相邻一站，不能跳站
  - 每回合必须移动，不能原地不动
  - 侦探之间不能占据同一站点

---
  游戏流程

  游戏共 24 回合。每回合执行顺序：

  1. Mr. X 先移动
  2. 然后所有侦探按顺时针依次移动

  Mr. X 的回合

  1. 选择一种交通方式移动到相邻站点
  2. 在移动记录板上写下目标站点编号
  3. 用一张交通票遮住站点编号——侦探只能看到他用了什么交通工具，看不到具体位置
  4. 将用掉的票交给公共票池（或给其他侦探使用）

  特殊：公开位置（Surfacing）

  Mr. X 必须在以下回合公开位置：第 3、8、13、18、24 回合

  公开时，Mr. X 将黑色棋子放在地图上对应站点。下一回合开始后可以移走棋子重新隐身。

  侦探的回合

  1. 侦探们可以互相商量策略
  2. 每位侦探选择交通方式移动棋子
  3. 将用掉的票交给 Mr. X 玩家

  重要：如果某侦探没有可用的票离开当前位置，该侦探被"困住"，棋子留在原地，本局无法再移动。

---
  Mr. X 的两种特殊票

  黑票（×5）

  - 可替代任何交通票（出租车/公交/地铁）使用
  - 侦探无法得知 Mr. X 使用了什么交通方式（看到的是黑票而非具体票种）
  - 还可以用于地图上泰晤士河上的渡船路线（黑色连线），这是 Mr. X 独有的移动方式

  双倍移动票（×2）

  - 使用后，Mr. X 在一回合内可以连续移动两次
  - 每次移动都要消耗对应的票（可以是不同类型）
  - 如果双倍移动的其中一次落在了公开回合，需在对应步骤后公开位置

---
  胜负条件

  侦探赢

  - 任何侦探移动到 Mr. X 当前所在站点，或 Mr. X 移动到某侦探所在站点——立即抓获，游戏结束
  - 所有侦探共享胜利

  Mr. X 赢

  - 坚持到第 24 回合结束仍未被抓
  - 所有侦探都被困住（无法移动）

---
  票的循环机制

  侦探用掉的票会交给 Mr. X 玩家。Mr. X 不会获得这些票来使用（他有自己的票），但这些票会进入公共票池供侦探们了解哪些票已
  被消耗。这个机制意味着随着游戏进行，特定交通方式的可用性会下降。
